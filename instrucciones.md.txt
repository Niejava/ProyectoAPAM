Proyecto APAM
APAM (Affective Plants Arduino Monitoring)

Realizar una interfaz web para monitorizar una planta.

La aplicación será parecida a la del clásico juego Tamagochi, pero con un avatar en la vida real.

Además de la interfaz web, también se ha de crear una base de datos de la cual se tome los valores de Temperatura, Humedad de la tierra, humedad del aire y luminosidad. 

La base de datos puede ser xml (con ExistDB por ejemplo) o Mysql.

Requisitos de herramientas técnicas:

- HTML5

- CSS3 con Bootstrap4

- Javascript

Además de la aplicación se pide una documentación de la misma, os adjunto una de otro grupo a modo de ejemplo.


Parsear un XML desde Javascript File
En la carpeta tenéis un ejemplo de una web simple, que carga un fichero xml y muestras los valores de esa BBDD-Xml en una tabla en la misma web.
